Overview

Passphrases generator created with JavaScript. Passphrases are created by using method called Diceware.

Features

  • Ability to generate passphrases in 12 languages.
  • Ability to select passphrase length (6-128 words).
  • Ability to generate new passphrase (using same settings) with just one button click.
  • Ability to copy generated passphrase with just one button click.

How it works

This website simulates throwing psychical dices (that are used in original version) 5 times. By doing that we get set of numbers which looks something like that: '11111', then we use it to get one world from World List array for language that user selected (for '11111' it's 'a' in English version) and we repeat that process as many times as selected length of passphrase to generate.
