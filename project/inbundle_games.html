Overview

Simple and fast tool to check if game was ever in a bundle at humblebundle.com. Created with React and PostgreSQL database + Node.js backend hosted on Heroku.

How it works

After entering the game title and pressing the enter button, the user is taken to the view with all games with similar titles - here we use RAWG API. Then, after pressing the mouse button on any game, the user is taken to the view with details about it. Where, behind the scenes, we send two fetches:

  • First to RAWG API to get screenshots and more details about this game.
  • And last to our backend - where with Node.js we search through the database for a game with the tittle we passed. If it is in the database, we return all its occurrences in it and by using the relationships we also return information about the bundles in which the game appeared.
