Overview

Tool for Spotify users, built with React, Spotify API and with backend hosted on Heroku.

Features

  • Log in with Spotify.
  • Option to display public and private playlists of user + information about them (num. of songs, name and cover art).
  • Display 100 songs (sorted by latest or oldest) in playlist.
  • Duplicate playlists with new number of songs, edit their title and description.
  • Deleting playlist from library with one button click.

How it works

After user successfully logged in with his account, we use Spotify API endpoint to GET list of all playlists of a current user and then we display information about all of them (name, num. of songs, cover art). By clicking on cover of a playlists we're redirect to subpage where more informations about that playlist are displayed + 100 (oldest or latest) songs from that playlist are displayed on that page too. By clicking 'Pare Down' button pop-up shows up where we can set up whole process of Pare Down - name of new playlist, description, privacy setting and number of songs. By clicking big and blue 'Pare Down' button on last step - new and pared down version of that playlist is created (by sending POST to Spotify API playlists endpoint).
