Overview

Completely free extension for Firefox, built with WebExtensions API - JavaScript, HTML and CSS. It changes new tab page to speed dial.

Features

  • Dark mode.
  • Built-in themes.
  • Organize bookmarks into groups.
  • Bookmark any website.
  • Synchronization between devices.
  • Minimalistic look.

How it works

By using WebExtensions API we can get all bookmarks that're stored in specific bookmarks folder (in this case: "simplestart"). Then we just create speed dial card for every one of them and then display them on homescreen.
